 To start game - open GameplayScene
 
 To configurate game settings - open Configuration.asset
 
 Tested with Unity 2021.3.3f
 
 &nbsp;
 
 Default key bindings:
 
 W - Move forward
 
 A - Turn left
 
 D - Turn right
 
 Spacebar - Shoot with bullet gun
 
 Left Control - Shoot with laser gun
